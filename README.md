# Certificate Generator (Java + PDFBox)

A simple and customizable PDF certificate generator written in Java.  
It uses a background certificate template image and places the recipient's details (name, course, date, ID) at configured coordinates on an A4 landscape PDF.  
Also includes a calibration mode and optional GUI.

---

## Features

- Uses an **A4 landscape certificate template** (PNG/JPG)
- Supports **Name**, **Course**, **Date**, and **Certificate ID**
- Text placement controlled by configurable coordinates
- Embeds custom **TrueType fonts** (UTF-8 / Unicode supported)
- Batch generation from **CSV**
- Optional **Swing GUI** for non-technical users
- Calibration mode to measure & fine-tune text positions
